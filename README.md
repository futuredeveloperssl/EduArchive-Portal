# EduArchive Portal

A modern, secure Learning Management System (LMS) for archiving and viewing educational content.

## Features
- **Secure Video Player**: Powered by YouTube IFrame API with referrer policy protection.
- **Cloud Database**: Integrated with Firebase Firestore for real-time progress and access control.
- **Admin Dashboard**: Easy lesson management and student approval queue.
- **Responsive Design**: Optimized for all devices with a premium glassmorphic UI.
